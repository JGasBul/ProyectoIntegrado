CARACTERÍSTICAS

Esta aplicación ha sido desarrollada mediante el lenguaje de programación Java. En concreta hemos utilizado la aplicación Eclipse que es una plataforma de software compuesto por un conjunto de herramientas de programación de código abierto.
La base de datos de la aplicación se encuentra en el servicio de AWS (Amazon Web Service). Es una colección de servicios de computación en la nube pública (también llamados servicios web) que en conjunto forman una plataforma de computación en la nube, ofrecidas a través de Internet por Amazon.com.
La base de datos ha sido diseñada con SQL, en concreto con la aplicación Heidi.En concreto, HeidiSQL es un software libre y de código abierto que permite conectarse a servidores MySQL.
